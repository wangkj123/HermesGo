# 002-便携构建步骤与后续工作

## 当前脚本做的事

1. 下载并解包官方 embeddable Python
2. 复制 Hermes 依赖
3. 复制官方 Hermes 源码
4. 生成新的 `HermesGo.bat`、`Setup-Ollama.bat`、`Verify-HermesGo.bat`
5. 生成默认的本地模型配置
6. 生成 `HermesGo.exe` 的马头图标
7. 把最终绿色包固定输出到 `create_hermes_go/output/HermesGo`
