# 002-便携构建步骤与后续工作

## 当前构建流程

1. 下载并解包官方 embeddable Python
2. 复制 Hermes 运行时依赖
3. 复制当前 Hermes 源码
4. 生成 `HermesGo.bat`、`Setup-Ollama.bat`、`Verify-HermesGo.bat`
5. 生成默认本地模型配置
6. 生成 `HermesGo.exe` 图标和经典启动器
7. 把最终绿色包输出到 `create_hermes_go/output/HermesGo`

## 当前已覆盖的能力

- 绿色版 / U 盘版 / 一键安装版交付

- Cloud: GPT-5.4 Mini 自动登录
- Hermes 自带的 Codex 认证链路
- tutorial 截图和录屏
