# 001-当前状态与标准边界

## 当前判断

- 当前能跑起来的是官方 Hermes v0.10.0 代码和官方 Dashboard。
- `create_hermes_go` 的目标是把这条工作链固化成脚本，并生成新的独立目录。
- 当前输出包的 Python 运行时来自官方 embeddable package，而不是本机系统 Python。

## 标准部分

- `python -m hermes_cli.main`
- `dashboard`
- `runtime/hermes-agent/web_dist`

- 便携 Python
- Windows bat 启动链
