# 001-当前状态与标准边界

## 当前判断

- 当前可交付的主体是 HermesGo 绿色版 / U 盘版 / 一键安装版。
- 这条线的目标是把 Hermes Agent、Dashboard、启动器和本地模型一起打包成可直接分发的 Windows 绿色包。
- 当前输出目录是 `create_hermes_go/output/HermesGo`，复制整个目录即可离线运行。
- 当前的 release 线不依赖本机安装的 Python、Ollama 或外部 Codex CLI。

## 标准部分

- `python -m hermes_cli.main`
- `dashboard`
- `runtime/hermes-agent/web_dist`
- `HermesGo.exe`
- `HermesGo.bat`
- `Verify-HermesGo.bat`
