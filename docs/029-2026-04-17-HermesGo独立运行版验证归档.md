# 029-2026-04-17-HermesGo独立运行版验证归档

## 时间

- 2026-04-17 11:24

## 目标

- 把 `create_hermes_go/output/HermesGo` 收敛到真正的独立运行包
- 不依赖系统 Python
- 输出包内能直接说明 Python 运行时从哪里获取

## 本轮修改

- 重写 `create_hermes_go/Create-HermesGo.ps1`
  - 用官方 `embeddable Python` 取代复制本机 Python 安装目录
  - 下载顺序固定为：国内镜像 `sdu-python-mirror` -> 国外官方直连 `python.org` -> 最后代理回退
  - 下载成功后把真实来源写入缓存旁路元数据和输出包 `runtime/python311/SOURCE.txt`
  - 生成隔离的 `python311._pth`，只保留：
    - `python311.zip`
    - `.`
    - `Lib/site-packages`
    - `../hermes-agent`
  - 删除 `site-packages` 中的 `__editable__*` 残留
  - `-Clean` 时先停止输出目录下的旧进程，避免 DLL 占用导致重建失败
- 增强生成出的 `Start-HermesGo.ps1`
  - 启动前清空 `PYTHONHOME` 和 `PYTHONPATH`
  - 聊天窗口启动命令同样清空这两个变量
- 增强生成出的 `Verify-HermesGo.ps1`
  - 先停止旧的 9119 监听
  - 用输出包自己的 `python.exe` 先做导入探测
  - 检查 `sys.path` 不含系统 Python 安装目录
  - 检查 9119 监听进程的 `ExecutablePath` 必须是输出包自己的 `python.exe`
