# 004-本地模型切换

## 目的

HermesGo 默认保持 `gemma:2b` 作为本地离线模型，但在需要时可以切换到另一个本地 Ollama 模型。

## 切换方式

1. 在 GUI 启动器里点“切换本地模型”，从已安装模型列表里直接选一个
2. 直接双击 `HermesGo\Switch-HermesGoModel.bat`
3. 或运行 `HermesGo\Switch-HermesGoModel.ps1 -Model <模型名>`
4. 或打开启动器，把 `Model` 改成目标模型后保存

## 说明

- 这个切换工具只改本地配置，不会在运行时下载模型。
- 目标模型必须已经在本地 Ollama 模型仓里存在。
- GUI 按钮会标出“可用 / 缺失”，缺失项也可以选择。
- 选择本地模型时会同步把 `provider` 切回 `ollama`，并恢复本地 `base_url`。
- 缺失模型会在启动 HermesGo 时通过包内 Ollama 下载到 `data/ollama/models`，不会写入系统 Ollama 目录。
- 切回默认值时，把模型名改回 `gemma:2b` 即可。

## 影响的文件

- `home/portable-defaults.txt`
- `home/config.yaml`
