# data

This folder stores local data used by the Docker-backed HermesGo setup.
这个目录保存 Docker 版 HermesGo 使用的本地数据。

## Subdirectories / 子目录

- `hermes/`
  - Hermes configuration, auth, sessions, and state / Hermes 配置、认证、会话和状态
- `ollama/`
  - Ollama model cache and persistent data / Ollama 模型缓存和持久化数据

Copy the whole folder to move the Docker-backed environment.
搬移 Docker 版环境时，直接复制整个目录。
