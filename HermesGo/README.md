# HermesGo

HermesGo is the Windows launcher for this repository. The main entry point is `HermesGo.exe`.

## Start

1. Double-click `HermesGo.exe`.
2. Use the launcher to start Hermes, sign in to Codex, or adjust the model settings.
3. Read `docs/README.md` for technical and development details.

## Directory Overview

- `home/`: Launcher state, session data, and runtime files
- `runtime/`: Hermes runtime files
- `installers/`: Installer caches and download notes
- `tools/`: Build and maintenance scripts
- `docs/`: Technical and developer documentation
- `logs/`: Startup and verification logs
- `data/`: Persistent data for bundled services
