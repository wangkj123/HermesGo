# installers

This folder stores installer payloads and offline downloads.
这个目录保存安装包和离线下载文件。

## Contents / 内容

- `Pinokio/`
  - Windows Pinokio installer assets / Windows Pinokio 安装器资源
- `ollama-windows-amd64.zip`
  - Offline Ollama archive for manual extraction / 供手动解压的离线 Ollama 压缩包

## Why this folder exists / 这个目录的用途

- First-time setup uses local installers.
- Keeping downloads here supports reproducible rebuilds and offline use.
- 首次部署直接使用本地安装包。
- 下载文件放在这里，重建结果更稳定，也适合离线使用。
