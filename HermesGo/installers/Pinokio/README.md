# Pinokio installer / Pinokio 安装器

This folder stores the Windows Pinokio installer used by HermesGo.
这个目录保存 HermesGo 使用的 Windows Pinokio 安装器。

- Expected file name: `Pinokio.exe` / 期望文件名：`Pinokio.exe`
- HermesGo uses this file when it exists. / 文件存在时，HermesGo 直接使用它。
- HermesGo downloads the latest stable installer from the official Pinokio download page when the file is missing.
  / 文件缺失时，HermesGo 从官方 Pinokio 下载页获取最新稳定版安装器。
